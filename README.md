- **Class & Object** вЂ” Cetak biru dan instansiasi objek
- **Encapsulation** вЂ” Private attribute dengan getter/setter
- **Inheritance** вЂ” Pewarisan sifat dari class induk ke class anak
- **Polymorphism** вЂ” Satu method, banyak bentuk perilaku
- **Abstraction** вЂ” Abstract class sebagai kontrak/blueprint
